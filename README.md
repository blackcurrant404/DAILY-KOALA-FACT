# Daily Koala Fact 🐨

A web app that displays a daily koala fact.
The same fact is shown throughout the day and changes the next day.

Features

- Loads koala fact from a JSON file (daily-based)
- Background video (loop) and music (both decorative)

How it works:

When the user clicks the button on the website, it displays the fact from a previosly loaded JSON file.
